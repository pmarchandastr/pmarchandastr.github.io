Useful tools

Public codes

  • APE (Analytical Protostellar Environment)

    • A code to simulate the physical conditions in protostellar systems. Provides interfaces and wrapper of the chemical code Nautilus, the radiative transfer code RADMC-3D, and the imaging software Imager, for easier chemical simulations and synthetic observations.
    Reference: Marchand et al 2025


  • Ishinisan

    • A code that computes the coagulation and ionization of grains for an arbitrary size-distribution. You can use it to generate a table to use in simulations or post-process your simulation data.
    Reference: Marchand et al 2021


  • Chemical code

    • A code to calculate the chemical abundances and the fraction of ionisation in protostellar environment. You can generate a table of abundance and calculate non-ideal MHD resistivities.
    Reference: Marchand et al 2016
